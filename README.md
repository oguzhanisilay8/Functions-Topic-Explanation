### 3. Simple Function Example
- A basic function that multiplies a number by two
- How function arguments work
- How results are returned

### 4. Biological Example: GC Content Calculation
- Writing a GC content function using:
  - `for` loops
  - `if` conditions
  - counters (accumulators)
- Understanding each line of the function
- Applying the function to a DNA sequence

### 5. Bioinformatics Example: Leaf Status Analysis
- Using a function with `if / else`
- Returning categorical biological results
